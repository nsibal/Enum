How to Compile:
1. Unzip contents of nxs180002.zip into a folder named "nxs180002".
2. Run the command `javac nxs180002/*.java` to compile.
3. Run the command `java nxs180002.EnumSP11 <n> <choice>` to execute the program.

where,
n - Number of elements the array
choice - 1 or 2
	1 for Select Method
	2 for PQ Method
